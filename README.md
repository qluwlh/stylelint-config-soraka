# stylelint-config-soraka

包含 prettier，stylelint 的配置文件
# Use

安装

```bash
npm i stylelint-config-soraka --save-dev
yarn add stylelint-config-soraka -D
```

in `stylelint.config.js`

```js
module.exports = {
  extends: ['stylelint-config-soraka'],
  rules: {
    // your rules
  },
};
```
